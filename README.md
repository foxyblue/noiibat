</h4>

## Usage

Have [noiibat](https://bulbapedia.bulbagarden.net/wiki/Noibat_(Pok%C3%A9mon)) listen on `4` ports.

```bash
noiibat --listeners 4
```

Serve noiibat using config.

```bash
noiibat --config noiibat_config.yaml
```
